# PostStat
https://vk.com/app4538648

Анализирует все посты ВКонтакте друга, сообщества или по любому другому адресу. Можно задать время, за которое надо проанализировать посты.
После анализа выдает:
* Общую информацию (число постов, лайков, комментариев, репостов)
* ТОП постов по лайкам
* ТОП постов по комментариям
* ТОП постов по репостам
* ТОП авторов постов
* Первые записи
* Популярные репосты
